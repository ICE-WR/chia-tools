# ChiaTools

#### 特色功能
1. 可多任务同时P图
2. P图进度精细到1%
3. 可暂停P图任务，可任意控制多P图任务的进度
4. 使用HPool官方挖矿程序挖矿，程序崩溃或异常时会自动重启
5. 不索要助记词
6. 任务失败后可以删除临时文件
7. 当任务执行到最后一步（拷贝文件）前，自动执行下一个任务
8. 可以修改任务，在下一个任务执行时生效
